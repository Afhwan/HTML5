# Modul 05: Elemen Semantik

Modul ini membahas tentang pentingnya menggunakan tag yang memiliki makna (semantic) daripada hanya menggunakan `<div>` atau `<span>`.

## Mengapa Semantik Penting?

1.  **SEO (Search Engine Optimization)**: Membantu mesin pencari memahami struktur dan konten penting di website Anda.
2.  **Aksesibilitas (Accessibility)**: Membantu alat pembaca layar (screen reader) bagi penyandang disabilitas dalam menavigasi situs.
3.  **Keterbacaan Kode**: Membuat kode lebih mudah dipahami oleh pengembang lain.

## Elemen Utama HTML5

- `<header>`: Bagian kepala halaman atau artikel.
- `<nav>`: Bagian khusus untuk link navigasi.
- `<main>`: Konten utama yang unik bagi dokumen tersebut.
- `<article>`: Konten mandiri (seperti berita atau blog).
- `<section>`: Bagian dari dokumen yang memiliki tema tertentu.
- `<aside>`: Konten sampingan atau tambahan.
- `<footer>`: Bagian kaki halaman atau artikel.

## Contoh Struktur

```html
<header>
  <h1>Logo</h1>
  <nav>...</nav>
</header>
<main>
  <article>...</article>
</main>
<footer>...</footer>
```
