# Teks dan Format HTML

Halaman ini mendemonstrasikan berbagai cara untuk memformat teks dan membuat daftar di HTML.

## Konsep yang Dipelajari

### Headings

- Penggunaan tag `<h1>` sampai `<h6>` untuk membuat judul dan subjudul dengan berbagai tingkatan kepentingan.

### Paragraf

- Penggunaan tag `<p>` untuk mendefinisikan blok teks atau paragraf.

### Daftar (Lists)

- **Unordered List (`<ul>`)**: Daftar tanpa urutan tertentu ( bullet points). Menggunakan tag `<li>` untuk setiap item.
- **Ordered List (`<ol>`)**: Daftar berurutan (angka atau huruf). Atribut `type` dapat digunakan untuk mengubah jenis penomoran (misal: `type="A"`).

### Format Teks

- `<b>` & `<strong>`: Menampilkan teks tebal.
- `<i>` & `<em>`: Menampilkan teks miring atau penekanan.
- `<u>`: Menandai teks dengan garis bawah.
- `<mark>`: Menyoroti teks.
- `<small>`: Mengecilkan ukuran teks.
- `<del>`: Menampilkan teks yang dicoret.
- `<ins>`: Menampilkan teks sebagai sisipan (garis bawah).
- `<sub>` & `<sup>`: Menampilkan subscript (bawah) dan superscript (atas), berguna untuk rumus kimia atau matematika.
