# Modul 03: Tautan dan Gambar

Modul ini membahas cara menghubungkan satu halaman ke halaman lain serta cara menampilkan gambar di web.

## Apa yang dipelajari?

1.  **Tag Tautan (`<a>`)**:
    - `href`: Menentukan alamat tujuan.
    - `target="_blank"`: Membuka tautan di tab baru.
2.  **Tag Gambar (`<img>`)**:
    - `src`: Lokasi file gambar.
    - `alt`: Teks alternatif untuk aksesibilitas.
3.  **Fragment Identifier (#ID)**:
    - Membuat navigasi di dalam satu halaman yang sama.

## Contoh Penggunaan

```html
<a href="https://google.com">Klik di sini</a>
<img src="gambar.jpg" alt="Pemandangan" />
```
