# Modul 04: Tabel dan Form

Modul ini menjelaskan cara mengelola data dalam bentuk tabel dan cara membuat input interaktif bagi pengguna melalui formulir.

## Apa yang dipelajari?

1.  **Struktur Tabel**:
    - `<table>`: Kontainer utama.
    - `<tr>`: Table Row (Baris).
    - `<th>`: Table Header (Judul kolom).
    - `<td>`: Table Data (Isi sel).
2.  **Elemen Formulir**:
    - `<form>`: Kontainer input.
    - `<input>`: Berbagai tipe seperti `text`, `email`, `checkbox`, `radio`.
    - `<textarea>`: Input teks panjang.
    - `<label>`: Label untuk setiap input (penting untuk aksesibilitas).
    - `<button>`: Tombol untuk submit atau reset.

## Contoh Form Sederhana

```html
<form>
  <label>Nama:</label>
  <input type="text" name="nama" />
  <button type="submit">Kirim</button>
</form>
```
