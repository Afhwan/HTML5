# Pengenalan HTML

Halaman ini berisi dasar-dasar struktur dokumen HTML5.

## Konsep yang Dipelajari

- **`<!DOCTYPE html>`**: Mendeklarasikan bahwa dokumen ini adalah HTML5.
- **`<html lang="id">`**: Root element dari halaman HTML dengan atribut bahasa Indonesia.
- **`<head>`**: Bagian yang berisi metadata tentang dokumen.
- **`<meta charset="UTF-8">`**: Menentukan encoding karakter untuk dokumen (UTF-8).
- **`<meta name="viewport">`**: Mengatur tampilan agar responsif di berbagai perangkat.
- **`<title>`**: Judul halaman yang muncul di tab browser.
- **`<body>`**: Konten utama halaman yang akan ditampilkan kepada pengguna.
