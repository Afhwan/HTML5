Pengenalan HTML5

HTML (HyperText Markup Language) adalah bahasa standar yang digunakan untuk membuat struktur halaman web. Versi terbaru saat ini adalah HTML5.

Struktur Dasar Dokumen HTML

Setiap file HTML memiliki struktur dasar sebagai berikut:

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Judul Halaman</title>
</head>
<body>
    <!-- Konten di sini -->
</body>
</html>
  • <!DOCTYPE html>: Mendeklarasikan bahwa dokumen ini menggunakan HTML5.
  • <html>: Elemen akar (root) dari halaman.
  • <head>: Berisi informasi meta tentang dokumen (tidak terlihat oleh pengunjung).
  • <body>: Berisi semua konten yang terlihat oleh pengunjung.
