# Sejarah Singkat HTML

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat halaman web. Berikut adalah perjalanan singkatnya.

## Awal Mula (1989–1991)
- **1989**: **Tim Berners-Lee**, ilmuwan di CERN, mengusulkan sistem informasi berbasis hypertext untuk berbagi dokumen penelitian.
- **1990**: Ia menciptakan tiga teknologi fundamental:
  - **HTML** (dengan 18 elemen awal)
  - **Browser/editor** pertama (bernama *WorldWideWeb*)
  - **Server web** pertama
- **1991**: Dokumen publik pertama tentang tag HTML dirilis.

## Perkembangan Versi HTML

| Versi | Tahun | Fitur Utama |
|-------|-------|-------------|
| **HTML 1.0** | 1993 | Versi informal, dasar struktur web. Browser *Mosaic* mulai populer. |
| **HTML 2.0** | 1995 | Standar resmi pertama (IETF). Memperkenalkan **formulir**. |
| **HTML 3.2** | 1997 | Dirilis W3C. Menambahkan **tabel**, **applet**, dan teks mengelilingi gambar. |
| **HTML 4.0** | 1997 | Memisahkan struktur (HTML) dan tampilan (CSS). Elemen presentasi (`<font>`) mulai ditinggalkan. |
| **HTML 4.01** | 1999 | Perbaikan dari 4.0. Menjadi standar paling stabil dan lama digunakan. |

## Masa Transisi: XHTML (2000–2008)
- **XHTML 1.0 (2000)**: HTML 4.01 ditulis ulang dengan aturan XML yang lebih ketat (tag harus ditutup, huruf kecil semua).
- **XHTML 1.1 (2001)**: Modular, semakin ketat. Banyak pengembang menganggapnya terlalu kaku, sehingga adopsi terbatas.

## Era Modern: HTML5 (2008–Sekarang)
- **2004**: **WHATWG** (Apple, Mozilla, Opera) dibentuk karena frustrasi dengan lambatnya W3C. Mereka mulai mengembangkan spesifikasi praktis untuk aplikasi web.
- **2008**: Draf pertama **HTML5** dirilis.
- **2014**: HTML5 resmi menjadi standar W3C.

### Fitur Utama HTML5:
- **Semantik**: Elemen struktural baru (`<header>`, `<footer>`, `<article>`, `<section>`).
- **Multimedia native**: `<audio>` dan `<video>` tanpa plugin.
- **Grafis**: `<canvas>` untuk menggambar 2D dengan JavaScript.
- **API canggih**: Penyimpanan lokal (*localStorage*), geolokasi, *web workers*.
- **Kompatibel** dengan konten lama dan lebih toleran terhadap kesalahan kode.

## Kesimpulan
HTML berevolusi dari alat sederhana untuk dokumen ilmiah menjadi fondasi utama web modern. Dengan HTML5, ia mampu mendukung aplikasi web yang dinamis, interaktif, dan kaya fitur tanpa plugin tambahan.
