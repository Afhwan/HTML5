# Modul 06: Multimedia

Modul ini menjelaskan cara menyisipkan konten multimedia seperti suara, video, dan konten dari situs luar (seperti YouTube) ke dalam halaman web.

## Apa yang dipelajari?

1.  **Elemen `<audio>`**:
    - Atribut `controls`: Menampilkan tombol play, pause, dan volume.
    - Elemen `<source>`: Menentukan sumber file audio.
2.  **Elemen `<video>`**:
    - Hampir sama dengan audio, tapi menampilkan visual. Atribut lain seperti `width` dan `height` bisa ditambahkan.
3.  **Elemen `<iframe>`**:
    - Digunakan untuk menyisipkan dokumen HTML lain ke dalam halaman saat ini. Sering digunakan untuk embed Video YouTube atau Peta Google.

## Tips Ekstra

Pastikan Anda menggunakan format file yang umum didukung:

- Audio: MP3, WAV, OGG.
- Video: MP4, WebM, OGG.
