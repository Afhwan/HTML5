Multimedia dalam HTML5

HTML5 memudahkan integrasi audio, video, dan konten eksternal tanpa memerlukan plugin tambahan.

Audio

Gunakan tag <audio> dengan atribut controls.

(Catatan: File audio harus tersedia di server Anda)

Video

Gunakan tag <video>.

Konten Eksternal (Iframe)

Gunakan <iframe> untuk menyisipkan halaman lain, seperti video dari YouTube.
